# 학습목표
1. 역할(persona) 부여를 통한 답변 생성
2. zeroshot / oneshot / fewshot prompting 이해
    - shot : 예시
    - zeroshot : 모델에 구체적인 예시를 제공하지 않고 주어진 작업을 수행하라고 요청하는 것. 모델의 일반적인 지식과 문맥 이해력을 기반으로 답변.
    - oneshot : 하나의 예시를 제공하고 모델이 이를 참고해 작업을 수행하도록 하는 것. 모델에게 기대하는 답변의 스타일을 알려줌으로써, 사용자가 원하는 답변을 유도.
    - fewshot : 2~5개의 예시를 제공하는 방식. 모델이 다양한 예시를 통해 작업의 맥락을 더 잘 이해하고, 보다 정확하게 작업을 수행할 수 있도록 유도.
